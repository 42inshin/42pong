# 42pong

이 프로젝트는 실시간 멀티 Pong 게임을 위한 SPA 웹사이트입니다.
42 OAuth 사용하여 로그인하며, 이중 인증, 프로필 사진 업로드, 닉네임 변경, 채팅 등의 기능이 있습니다.


## 기술 스택
- Vue.js (composition API, Pinia)
- TypeScript
- HTML / CSS / JavaScript
- Nest.js
- PostgreSQL
- Vite
- Prettier
- Docker(Docker-compose)


## 실행

```bash
make
# or
docker-compose up --build
```
